Flash Cards Stylées - Espaces Préhilbertiens et Euclidiens

Qu'est-ce qu'un produit scalaire (ps) sur un espace vectoriel réel E ?
Un produit scalaire vérifie :
1. Symétrie : ∀x,y ∈ E, <x,y> = <y,x>
2. Bilinéarité : ∀x,y,z ∈ E, ∀λ,μ ∈ ℝ, <λx + μy, z> = λ<x,z> + μ<y,z> et <z, λx + μy> = λ<z,x> + μ<z,y>
3. Positivité : ∀x ∈ E, <x,x> ≥ 0
4. Définitude : <x,x> = 0 ⇒ x = 0E
Définition d'un espace préhilbertien réel.
E est un ℝ-espace vectoriel muni d'un produit scalaire. La dimension peut être finie ou infinie.
Qu'est-ce qu'un espace euclidien ?
Un espace préhilbertien de dimension finie.
Norme associée à un produit scalaire.
∀x ∈ E, ||x|| = √<x,x>. C'est la norme du vecteur x.
Distance associée au produit scalaire.
∀x,y ∈ E, d(x,y) = ||x - y||.
Identités remarquables pour les normes.
1. ||x + y||² = ||x||² + ||y||² + 2<x,y>
2. ||x - y||² = ||x||² + ||y||² - 2<x,y>
3. Identité du parallélogramme : ||x + y||² + ||x - y||² = 2(||x||² + ||y||²)
Inégalité de Cauchy-Schwarz.
∀x,y ∈ E, |<x,y>| ≤ ||x|| ||y||. Égalité si x et y sont colinéaires.
Propriétés de la norme.
- ∀x ∈ E, ∀λ ∈ ℝ, ||λx|| = |λ| ||x||
- ∀x,y ∈ E, ||x + y|| ≤ ||x|| + ||y|| (inégalité triangulaire)
- ||x|| = 0 ⇔ x = 0
Définition de l'orthogonalité.
Deux vecteurs x et y sont orthogonaux si <x,y> = 0.
Famille orthogonale.
Famille de vecteurs deux à deux orthogonaux. Toute famille orthogonale sans le vecteur nul est libre (linéairement indépendante).
Théorème de Pythagore dans un espace préhilbertien.
1. Pour une famille orthogonale {e1, ..., en}, ||e1 + ... + en||² = ||e1||² + ... + ||en||²
2. Pour deux vecteurs x,y : <x,y> = 0 ⇔ ||x + y||² = ||x||² + ||y||²
Famille orthonormale.
Famille orthogonale où chaque vecteur a une norme égale à 1. Pour {e_i}, <e_i, e_j> = δ_{ij} (1 si i=j, 0 sinon).
Vecteur unitaire.
Vecteur de norme égale à 1.
Théorème d'orthonormalisation de Gram-Schmidt.
Soit {e1, ..., en} une famille libre de E. Il existe une unique famille orthonormale {ε1, ..., εn} telle que :
- ∀k, Vect(e1,...,ek) = Vect(ε1,...,εk)
- ∀k, <εk, ek> > 0
Formule : ε1 = e1 / ||e1||, puis εk = (ek - ∑_{i=1}^{k-1} <ek, εi> εi) / ||ek - ∑_{i=1}^{k-1} <ek, εi> εi||
Orthogonal d'un sous-ensemble F (F⊥).
Ensemble des vecteurs de E orthogonaux à tous les vecteurs de F. F⊥ est un sous-espace vectoriel de E.
Projection orthogonale sur un sous-espace de dimension finie F.
Pour x ∈ E, il existe un unique Pf(x) ∈ F tel que :
- Pf(x) ∈ F
- x - Pf(x) ∈ F⊥
Si {ε1, ..., εn} base orthonormale de F, Pf(x) = ∑_{i=1}^n <x, εi> εi
Décomposition orthogonale.
E = F ⊕ F⊥ (somme directe). ||x||² = ||Pf(x)||² + ||x - Pf(x)||²
Propriétés de l'application p(f).
p(f) : E → E est un projecteur (p(f)² = p(f)), de noyau F⊥ et d'image F. C'est le projecteur orthogonal sur F.
Distance d'un vecteur x à un sous-espace F de dimension finie.
d(x, F) = ||x - Pf(x)||
